RoMatch

RoMatch is a module for easy matchmaking on Roblox.

Learn more

What is RoMatch?

RoMatch is a module that handles matchmaking on Roblox. It is customizable and easy to use, but can be changed by advanced users. RoMatch has frequent updates that add new features such as cross-server matchmaking.

Easy to install and use

RoMatch is easy to integreate and change. It handles all lobbies for you so you can focus on your game.

Modular

You can easily contribute and change the module. It also supports having multiple game modes in the same server.

Cross-server Matchmaking

RoMatch allows for cross-server matchmaking using an external server over HTTP (currently unavaliable).
